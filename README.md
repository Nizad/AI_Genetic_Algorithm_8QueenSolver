# 8-Queen Solver with Genetic Algorithm
## Description
Solving the problem of 8 Queens:
In the 8-Queen problem, 8 ministers should be placed on an 8 x 8 chess board, so that none of the two queens threaten each other.

## Technologies
- Genetic Algorithm
- Matlab
